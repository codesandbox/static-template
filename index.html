Integrador Programación II

El siguiente exámen integrador, tendrá por objetivo poder utilizar las herramientas aprendidas a lo largo del año en Programación. Aunque cabe destacar que estas mismas herramientas las utilizamos siempre en nuestra práctica, hablamos de HTML, CSS y JavaScript.
De todas formas, acá podemos ver un listado mas detallado:

  • Estructura documento HTML
  • CSS: Selectores y edición de estilos
  • JavaScript: Variables, Condicionales, Repetición y DOM
El exámen sólo será práctico, en este punto nos vamos a enfocar en poder utilizar bien las herramientas vistas en clase.

Práctico

El problema es el siguiente, tenemos un sitio web de una famosa pizzeria de santa rosa, pero los estilos no cargan y no funciona algunas de las opciones básicas, tendremos que arreglar el sitio para que funcione correctamente

  1. Corregir el documento html para que puedan cargar los estilos y el script de JavaScript
  2. A los estilos agregar:
    1. Nombre de la empresa.
    2. Menu seleccionado.
    3. Foto de las variedades de pizza junto a su descripción
    4. Editar el documento con css.
  3. Agregar botones que cambien las opciones del menu, mostrando otros tipos de pizza junto a su descripción
  4. Agregar un boton, donde muestre el costo de las pizzas en un alert() o console.log().
  5. Siguiendo el ejercicio anterior, vamos a mejorarlo. El usuario debe ingresar el nombre de la variedad que quiere, y mostrar el precio en un alert() o console.log().
  6. Crear una función que cargue distintos sabores de pizza en un arreglo, y que el usuario decida cuando deja de cargar o no.
