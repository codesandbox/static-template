Instructions

1. Tap the numbers on the grid to add them upto the target value.
2. You can earn points for each number used to form the target number.
3. If you exceed your target, the selected numbers are reset and you must start again.
4. A new row of numbers are spawned at the top when the timer runs out.
5. The game ends when the timer ends and there is no longer any space in the bottom row.
