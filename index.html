    Take Home Activity

  • Instructions can be found here
      • User Story In a lot of websites, you’ll find a Meet the Team page where the company showcases the core team members along with their role and a short bio or description which is what we'll be creating

    1. Read more about wireframes in this article
    2. Create an HTML page with the usual declarations
    3. Define CSS internally (inline CSS not recommended)
        4. ‘team-member’ class style specifications
      • Each section should have a fixed width of 256px. (Ref: Width )
      • Each paragraph under the ‘team-member’ class should have an opacity = 0.8.
    4. Use semantic tags like header and section to define parts of your HTML code.
    5. Add a title = “Meet the team” to the page.
    6. The page body should be “snow” in color.
    7. Add a header with the following elements
      1. A main heading saying “Meet the team” with the class as “main-heading”
      2. A subtext under the heading saying - “Meet the people who make this software awesome”. Use the class “main-subtitle” to style this text
      3. Use the following styles
        • Heading should be indigo in color.
        • The paragraph should be gray in color.
    8. Add three sections (corresponding to three team members) with class = ‘team-member’ and id equals the name of the member in lowercase. Example if the members name is “Vivek Nigam” the id of their section should be “vivek”.
      1. Each element of the ‘team-member’ class should have the following children.
      2. An image of width and height 100 with alt text as the member’s name. This shall serve as the avatar of the person.
      3. An h2 heading to display the name
      4. An h3 heading to display the role
      5. A paragraph text to display the bio.
    9. The styles for each section of the class ‘team-member’ will be different and are provided in Appendix.

      10. Solution
