C++:Math



C++ offers some basic math functions and the required header file to use these functions is <math.h>.

Max and min

The max(x,y) function can be used to find the highest value of x and y:
cout << max(5, 10);
And the min(x,y) function can be used to find the lowest value of x and y:
cout << min(5, 10);

C++ <cmath> Header

  1. abs(x)
    • Returns the absolute value of x

  2. acos(x)
    • Returns the arccosine of x

  3. asin(x)
    • Returns the arcsine of x

  4. atan(x)
    • Returns the arctangent of x

  5. cbrt(x)
    • Returns the cube root of x

  6. ceil(x)
    • Returns the value of x rounded up to its nearest integer

  7. cos(x)
    • Returns the cosine of x

  8. cosh(x)
    • Returns the hyperbolic cosine of x

  9. exp(x)
    • Returns the value of E*

  10. expm1(x)
    • Return e*-1

  11. fabs(x)
    • Returns the absolute value of a floating x

  12. fdim(x,y)
    • Returns the positive difference between x and y

  13. floor(x)
    • Returns the value of x rounded down to its nearest integer

  14. hypot(x,y)
    • Return the sqrt(x*x+y*y) without intermediate overflow or underflow

  15. fma(x,y,z)
    • Returns x*y+z with losing precision

  16. fmax(x,y)
    • Returns the highest value of a floating x and y

  17. fmin(x,y)
    • Returns the lowest value of a floating x and y

  18. fmod(x,y)
    • Returns the floating point remainder of x/y

  19. pow(x,y)
    • Returns the value x to the power of y

  20. sin(x)
    • Returns the sine of x

  21. sinh(x)
    • Returns the hyperbolic sine of a doubke value
