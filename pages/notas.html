Listado de códigos (Eventos de analytics)

    >>Boton de Whatsapp<<

windows.addEventListener('DOMContentLoaded', function() {
 document.querySelector('#whatsapp-button').addEventListener('click', function (){
  gtag('event', 'whatsapp', {
  'event_category': 'botonClicado',
  });
 });
});

    >>Agregar al Carrito<<

window.addEventListener('DOMContentLoaded',function() {
 document.querySelectorAll('.add_to_cart_button').forEach(function(btn){
  btn.addEventListener('click',function(){
   gtag('event', 'adicionar-carrito', {
   'event_category': 'carrito',
   });
  });
 });
});

    >>Producto visto<<

windows.addEventListener('DOMContentLoaded', function() {
 if(window.location.href.includes('product')){
  gtag('event', 'vista_producto', {
  'event_category': document.querySelector('.product_title').innerText
  });
 };
});

    >>Compra con valor e Id de Transaccion<<

window.addEventListener('DOMContentLoaded',function() {
 if(window.location.href.includes('order-received')){
  gtag('event', 'compra', {
  'event_category': 'compra-realizada',
  'event_label': document.querySelector('.order_details .order strong').innerText
  'value': Number(document.querySelector('.total .amount').innerText.replace('R$','').replaceAll('.','').replace(',','.')),
  });
 };
});

Pasos a pasos importantes

    >>PASO A PASO PARA WEBSITE CALL CONVERSION<<

1-Entrar en herramientas y configuraciones.
2-Nueva acción de conversión.
3-llamadas telefónicas.
4-Llamadas a un número de teléfono en su sitio web.
5-Agregar nombre a la conversión.
6-Valor > no asignar ningún valor a esta acción de conversión.
7-Recuento > una.
8-Número de teléfono > seleccionar país > agregar No° teléfono.
9-Colocar los segundos para contabilizar la conversión.
10-Ventana de conversión post-clic > 30 días.
11-Modelo de atribución > Basada en datos Recomendada.
12-Crear y continuar.
13-Copiar y pegar el código generado dentro del sitio web.

>>Para comprobar si está funcionando una conversión de llamada en el sitio<<
Al final del enlace del sitio colocar #google-wcc-debug y selecciono Force en el cuadro que se habilita.
